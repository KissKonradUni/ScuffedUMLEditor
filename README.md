# Scuffed UML Editor

A web-based UML class diagram editor built with vanilla JavaScript and HTML5 Canvas.

## Features

- Create UML class diagrams with classes, comments, and relationships
- Visual canvas-based editing interface
- JSON-based project format
- Export diagrams as images
- Customizable canvas dimensions and colors

## Live Demo

Try it online: https://kisskonraduni.github.io/ScuffedUMLEditor

## Usage

Open `index.html` in a web browser to start creating UML diagrams. Use the editor panel to add classes, comments, and links between elements. Right-click on the canvas to save your diagram as an image.
